Inline

A seção "Inline" usa display: inline. Isso faz com que os elementos dentro dela se comportem como elementos de linha, ocupando apenas o espaço necessário ao seu conteúdo, sem criar um novo contexto de bloco.

Block

A seção "Block" usa display: block. Isso faz com que os elementos dentro dela se comportem como elementos de bloco, ocupando todo o espaço horizontal disponível e iniciando uma nova linha.

Inline-block

A seção "Inline-block" usa display: inline-block. Isso permite que os elementos dentro dela se comportem como elementos de linha, mantendo as características de bloco, como a possibilidade de definir largura e altura.

None

A seção "None" usa display: none. Isso faz com que a seção e seus elementos sejam completamente removidos do fluxo de layout, ou seja, eles não ocupam espaço na página e ficam invisíveis.
